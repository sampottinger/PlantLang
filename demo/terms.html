The Plant project takes open source and privacy seriously.

Summary

Everything is encrypted in transit when using HTTPS which is recommended. This website maintains regular server logs which include IP address and URL parameter. Raw logs kept for 3 days and aggregate statistical reports kept without default expiration. Your code is not sent back to the server but, if a URL is entered with code in it, that code is logged incidentally as part of the request URL. No third party analytics services are used (beyond the web host logs aggregation). See below for detailed information. This website is use at your own risk no warranty.
